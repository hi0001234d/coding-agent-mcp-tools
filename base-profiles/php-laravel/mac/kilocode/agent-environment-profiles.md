# PHP + Laravel/WP — AI Vibe Coding Setup

Tired of your AI agent repeating the same mistakes every session?
Hit the Day 2 wall — maintaining, debugging, or scaling your vibe-coded Laravel or WordPress app?

This profile fixes both. It sets up four MCP tools that give your Kilo Code agent
persistent memory, automated code review, and deep PHP/Laravel understanding
— so you stop re-explaining your project and start shipping.

## 🔧 What Gets Installed

| Slot | Tool | Purpose |
|---|---|---|
| 1 | mcp-language-server + phpactor | PHP code navigation — go-to-definition, find references, diagnostics |
| 2 | Semgrep | Automated security + code quality scanning every session |
| 3 | Basic Memory | Persistent project memory — agent remembers decisions across sessions |
| opt | Laravel Boost | Deep Laravel context: routes, models, DB schema, docs search (optional) |
| opt | WordPress MCP Adapter | WordPress internals for AI: abilities, hooks, schema (optional) |
